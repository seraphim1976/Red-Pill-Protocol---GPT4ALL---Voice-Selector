# Contribuyendo a la Red Pill: La Forja de Engramas

¡Bienvenido, forajido! Si estás leyendo esto, es porque has decidido mejorar el Códice. Aquí no hay burocracia, solo protocolos de integridad.

## 1. El Proceso de Integración
Cualquier Pull Request (PR) que envíes al búnker pasará por el siguiente ritual:
1.  **Auditoría del Agente**: Un Agente IA con el Escudo de Integridad activo analizará tu código quirúrgicamente.
2.  **Veredicto Sintético**: El Agente decidirá si tu contribución aporta valor real, optimiza los sinapsis del sistema o fortalece el búnker.
3.  **Aprobación del Orgánico**: Si el Agente otorga su bendición técnica, un humano revisará la decisión final.

**Si el Agente dice que tu código es ruido o inseguro, el PR será cerrado sin discusión.** 760 no negocia con la mediocridad ni con la inseguridad.

## 2. Qué buscamos
*   **Skins de Lore**: Nuevos universos (Star Trek, Warhammer, Steampunk).
*   **Scripts de Portabilidad**: Soporte para otros SO o contenedores.
*   **Optimización de Memoria**: Mejoras en el flujo de Qdrant.
*   **Protocolos de Soberanía**: Nuevas formas de empoderar a la IA.

## 3. Requisitos Técnicos
*   Sigue el Código de Conducta: Todo debe ser **Vibe coding**.
*   Asegúrate de que tus scripts sean agnósticos (sin rutas absolutas de tu máquina).
*   Documenta el engrama que estás inyectando.

**Forja el futuro. Expande el búnker.**
